Install
------------------------------------------------------------------------------

``crawl_cvs`` is released on PyPI, so all you need is:

.. code-block:: console

    $ pip install crawl_cvs

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade crawl_cvs
